Remilia OS 是一个面向操作系统竞赛与教学场景的 Rust 语言操作系统内核，以双架构统一设计和广泛的 Linux 兼容层为主要特色。项目核心代码约 4.6 万行，同时支持 RISC-V 64 和 LoongArch 64 架构，通过清晰的 arch 抽象层使近 84% 的代码保持架构无关。内核自研了内存管理（buddy + slab 分配器、写时复制页表、惰性映射与 VMA 合并）、三层时钟子系统、CFS 风格公平调度器（含 futex 与优先级继承）以及 Linux 风格中断域模型等基础组件，实现了超过 80 个真实系统调用，能够运行 BusyBox shell 和 libc 测例。项目在工程上展现出较强创新性，包括编译期嵌入 initramfs 以简化部署、使用 LR/SC 原子指令实现无锁多核自举、陷阱入口的哨兵字节重入检测，以及为比赛场景特化的内核内测试运行器。配合内核内单元测试、端到端集成测试和官方测例回归的三层测试体系，Remilia OS 在有限的代码规模内达到了令人瞩目的功能广度与工程成熟度，是一个兼具教学价值与竞赛竞争力的 Rust OS 实例。
