ZeroOS 是一个面向 OSKernel2024 比赛的 RISC-V 64 位 Linux 兼容宏内核操作系统。该项目采用纯 Rust 编写，基于 ArceOS 与 Starry 模块化框架构建，核心定位是在 RISC-V 平台上高效运行标准 Linux 用户态程序与测试套件。其核心特点包括实现了 101 个 Linux 系统调用，完整涵盖进程、内存、文件与网络子系统，并支持 ext4、FAT 等多种文件系统以及基于 smoltcp 的网络协议栈。ZeroOS 的主要亮点在于创新性地引入异步系统调用模型，利用 Rust 的 async/await 机制显著优化阻塞操作性能；同时具备出色的跨平台能力，不仅支持 QEMU 虚拟环境，还深度适配 VisionFive2 实体开发板，自主实现了 PLIC 中断控制器与 SD 卡驱动。此外，项目通过用户态映射巧妙解决了 FAT32 不支持符号链接的工程难题，展现了扎实的底层系统开发与工程实践能力。
