OSKernel C Base Model 是一个面向操作系统内核比赛的 RISC-V64 单内核项目，采用 C 语言和少量汇编实现，代码量约 1.5 万行。项目围绕 Sv39 虚拟内存、256 任务调度、155 个 Linux 兼容系统调用等核心子系统，构建了从启动加载 ELF 到用户态 Shell 运行的完整链路，技术定位为紧凑高效的类 Unix 教学/竞赛内核。

项目亮点包括：基于引用计数的 Copy-on-Write 内存管理，利用自定义页表标志位区分 COW 与 SysV 共享内存；execve 支持 musl/glibc 双 ABI 自动探测和路径重映射，兼具静态与动态链接 ELF 的加载能力；内存文件系统与 ext4 只读支持相结合，并提供文件缓存与名称缓存以适配比赛场景；用户态 init Shell 集成了管道、重定向、文本过滤和脚本编排等实用功能。多任务方面实现了克隆、信号递送与基于轮转的调度，设备驱动覆盖了 NS16550 串口输出、virtio-blk 只读和 virtio-net 原始帧收发。由于整体以较小的代码规模覆盖了引导、内存、进程、文件系统和设备驱动等多层能力，该项目在实用性、完整性和可比赛性上均有较好平衡。
