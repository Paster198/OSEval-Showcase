Eonix 是一个使用 Rust 编写的多架构宏内核操作系统，支持 x86_64、RISC-V 64 和 LoongArch64 三种主流指令集架构。其技术定位为一个面向教学与研究的现代化、高安全性操作系统内核。项目的核心特点在于深度利用 Rust 语言特性，创新性地采用 async/await 语法实现有栈与无栈混合的异步任务调度，并构建了类型安全的硬件抽象层。此外，内核实现了包含 Buddy 与 Slab 分配器及写时复制的完整内存管理、兼容 Linux clone 语义的进程模型以及支持多种文件系统的虚拟文件系统层。其最大亮点在于将异步运行时与 RCU 无锁数据结构巧妙结合，大幅提升了内核并发性能，同时通过自定义过程宏实现了跨架构的 Per-CPU 变量支持。该项目代码规模近四万行，整体功能完整度约 82%，充分展现了 Rust 在底层系统编程领域的强大能力与设计美学。
