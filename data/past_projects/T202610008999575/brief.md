ShellCore 是一个基于 Rust 语言开发的宏内核操作系统项目，定位为从教学系统向实用级别过渡的通用 OS 内核。它在 rCore-Tutorial v3 的基础上进行了全面扩展，共包含约 27,500 行 Rust 源码，实现了 159 个系统调用函数。

项目的核心特点包括：通过条件编译实现 RISC-V 64 与 LoongArch 64 双架构统一抽象，上层代码共享率约 90%；具备完整的 POSIX 信号框架，支持全部 64 种信号的嵌套处理；实现了 Ext4 文件系统的读取与部分写入能力，包括 extent 树的遍历和叶子节点插入；构建了四级优先级调度器，支持 FIFO、RR、OTHER、BATCH、IDLE 五种调度策略；提供 System V 消息队列与共享内存两种 IPC 机制，并支持命名空间隔离。

技术亮点包括：双层缓存架构（文件页缓存与块缓存）通过精心的锁序设计避免死锁；多粒度物理帧分配器支持 4KB/2MB/1GB 三种页大小的分配与回收；完整实现了 epoll、eventfd、futex、loop 设备等实用特性；基于 smoltcp 提供 TCP/UDP/RAW socket 网络支持。项目代码组织清晰，充分利用 Rust 的类型系统保障内存安全，已具备运行 BusyBox 及部分 LTP 测试套件的能力，体现了扎实的系统软件工程实现水平。
