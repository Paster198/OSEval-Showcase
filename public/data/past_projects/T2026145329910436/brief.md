StarryOS（VegarOS）是一个基于 ArceOS 组件化 unikernel 底座深度改造而成的 Linux 兼容宏内核。项目在复用 ArceOS 的硬件抽象、驱动框架和网络协议栈的基础上，自主实现了进程/线程管理、虚拟内存、信号处理、伪文件系统、TTY/PTY 子系统、System V IPC 等宏内核核心模块，并提供 211 条系统调用，可直接运行 BusyBox 与 LTP 测试套件，整体功能覆盖度约 78%。该系统支持 RISC-V 64、LoongArch 64 和 AArch64 三种架构，具备统一的文件描述符抽象层、基于引用计数的 CoW 内存映射、轻量级 VFS 框架和完整的 epoll 语义等工程亮点，通过离线构建方案适应竞赛环境。StarryOS 的核心创新在于用组件化 unikernel 组件搭建出类 Linux 宏内核，展现了在不牺牲 API 兼容性的前提下灵活复用已有基础框架的工程思路。
