Eureka OS 是一个面向竞赛场景的 Rust 宏内核项目，由武汉大学 Eureka 团队开发，同时支持 RISC-V 64 和 LoongArch 64 双架构。内核以 Linux ABI 兼容性为核心目标，实现了超过 200 项系统调用，广泛覆盖文件 IO、进程管理、内存管理、信号、网络、IPC、定时器、epoll 和 eBPF 等子系统，具备运行真实用户态工具链（含 musl/glibc）和 LTP 回归测试的能力。

其技术特点体现在几个层面。内存管理方面，内核提供了完整的虚拟内存、按需分页、写时拷贝（CoW）、大页/巨页、共享文件与匿名 mmap 以及 mlock/memfd 等机制，并借此支持高效的 fork 实现。文件系统部分以 EXT4 为主干，基于 lwext4 C 库构建了带页缓存、元数据缓存和预取的 Rust 抽象层，同时通过 VFS 接口集成了管道、epoll、eventfd、loop 设备和伪文件系统。在调度器设计上，项目采用基于 Future 的协程式包装与 per-CPU PELT 负载均衡相结合的方式，实现了优先级调度和初步的 SMP 支持。硬件抽象层通过统一的 trait 体系分离架构相关代码，使内核主体对 RISC‑V 与 LoongArch 的差异保持透明，并在 LoongArch 侧完整处理了 TLB 重填异常和双根页表切换等特有机制。

项目的亮点还包括：内建的多维度性能分析与内存泄漏诊断框架、支持 SysV 共享内存/消息队列/信号量的完整 IPC 子系统、软件 eBPF 解释器、以及围绕 LTP 构建的自动化评测体系。这些工程实践使其在大学生 OS 内核比赛中展现出较高的完成度和扎实的系统编程功底。
