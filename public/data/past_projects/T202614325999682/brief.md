Sustcore 是一个基于能力（Capability）的混合内核操作系统项目，采用 C++20/23 及少量 C 和汇编实现，代码量约 11.4 万行，支持 RISC-V64 与 LoongArch64 双架构。其核心特点在于以能力作为统一资源访问原语，所有内核对象（进程、线程、内存区域、文件、IPC 端点、通知、管道等）均通过能力空间中的槽位引用，配合 64 位细粒度权限位图和 CLONE/MIGRATE/MIGRATE_ONCE 等传递语义，实现最小权限原则下的安全资源共享。内存管理方面实现了 Buddy 页框分配器、SLUB 小对象分配器、带引用计数的 GFP 层以及支持懒分配、写时复制（COW）和文件后端映射的 Memory Payload 抽象，写时复制与能力克隆深度集成。VFS 子系统包含 ext4（从零实现读取与写入）、tmpfs、procfs、devfs 和 tarfs 五种文件系统以及基于 LRU 的页面缓存。任务管理提供进程树、线程模型、信号机制和异步等待/唤醒框架；多级调度器按优先级组织实时、轮转、先来先服务等调度类；IPC 通过端点支持同步/异步消息传递和 RPC 模式。项目在能力安全模型、ext4 实现、Memory Payload 设计及双架构对称支持等方面表现突出，展现出较高的技术完成度和创新性。
