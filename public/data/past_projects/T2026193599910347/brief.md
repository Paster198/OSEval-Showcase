Project Aurora 是一个基于 Rust 语言从零构建的 RISC-V64 单核宏内核，定位为面向 OS 竞赛的研究型系统。内核约 20,000 行代码，实现了 Sv39 分页、轮询调度器、92 个类 Linux 系统调用以及 VFS 统一挂载框架，能够运行交互式 shell、TCP/UDP echo、文件系统冒烟测试等多种用户态程序。其突出亮点包括：完全自主实现的 ext4 文件系统读写与 extent 树寻址、FAT32 支持以及配套的块缓存层；基于自定义 PTE_COW 标志与引用计数的 Copy-on-Write fork 机制；采用静态固定容量数组的无堆分配设计，结构清晰且可控；内嵌手写 RISC-V 机器码用户态测试程序，可在无外部工具链的情况下覆盖管道、futex、epoll、fork/execve/waitpid 等关键路径。网络栈基于 smoltcp 构建，提供 TCP/UDP/ICMP/ARP 支持，并通过 VirtIO MMIO 驱动与块设备和网卡交互。整体架构模块化，兼顾了内核核心子系统的覆盖广度与若干深入特性，展现出扎实的工程实现能力。
