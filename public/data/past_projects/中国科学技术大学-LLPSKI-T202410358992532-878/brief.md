StarsOS 是一个面向操作系统内核比赛的中等规模 RISC-V 64 位类 Unix 操作系统内核。该项目基于 C 语言与汇编语言开发，支持 SMP 多核启动，整体架构遵循经典 Unix 设计。其核心特点包括实现了高完整度的 FAT32 文件系统与 VFS 抽象层，具备基于 Sv39 三级页表的虚拟内存管理，并提供了约 85 个 Linux 兼容系统调用，涵盖进程、内存、文件与信号等核心子系统。项目的技术亮点在于引入了被动映射机制以高效实现按需调页，采用两层文件描述符设计优化进程派生性能，借鉴 Linux 的 initcall 机制实现内核模块自动注册，并设计了时间复杂度为 O(1) 的定时睡眠事件队列。整体代码结构清晰，在文件系统与内存管理方面展现出较高的工程完成度与设计创新性。
