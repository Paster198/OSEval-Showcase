PwnMyOS 是一个面向全国大学生计算机系统能力大赛操作系统设计赛的类 Linux 兼容内核，基于 Rust 语言开发，支持 RISC-V64 和 LoongArch64 双架构。该项目以 Linux 兼容性广度为设计重心，实现了超过 100 个系统调用，涵盖进程管理、文件系统、内存管理、信号处理等主要子系统，并提供完整的 POSIX 线程支持。核心特点包括：通过 lwext4 C 库与 Rust 的 FFI 绑定，直接获得功能完备的 ext4 文件系统支持；在 exec 路径中内建 busybox 智能路由与 LTP 测试兼容层，可同时适配 glibc 和 musl 编译的用户程序，显著提升比赛评测场景下的测试通过率；基于 polyhal 硬件抽象层实现 RISC-V 和 LoongArch 双架构的 virtio-blk 驱动适配；拥有进程组信号广播、robust futex、孤儿进程收养等健壮的进程管理机制。项目采用 FIFO 调度与单核设计，系统同步设施较为基础，网络栈以占位为主，但其务实的工程策略、扎实的系统调用覆盖与已验证的 QEMU 运行能力，使其成为一款实用导向的竞赛内核作品。
