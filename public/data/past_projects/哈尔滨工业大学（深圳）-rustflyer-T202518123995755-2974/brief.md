Nighthawk OS 是一个基于 Rust 语言编写的类 Linux 操作系统内核，主要面向 RISC-V 与 LoongArch 架构。该项目的技术定位在于探索现代系统编程语言在操作系统底层的深度应用。其核心特点是全面采用 Rust 的 async/await 无栈协程机制作为任务调度基础，消除了传统内核中复杂的上下文切换开销，并利用编译期检查保障并发安全。在功能实现上，Nighthawk OS 具备较高的工程完整度，实现了约 192 个系统调用，涵盖进程管理、Sv39 虚拟内存、VFS 及网络栈等核心子系统。其突出亮点包括：设计了基于函数指针的零开销模块化 VMA 页错误处理机制；实现了 epoll、inotify、timerfd 等丰富的 Linux 特殊文件系统以支持复杂用户态程序；以及通过统一的宏抽象实现了跨架构的代码复用。总体而言，该项目在架构创新与 POSIX 兼容性方面展现了出色的技术水平。
