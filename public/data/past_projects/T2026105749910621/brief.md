F423OS 是一个基于 xv6-riscv 改造的竞赛型操作系统内核，核心目标是在不实现完整通用 OS 的前提下，精确通过初赛各项 Linux ABI 兼容性测试。项目在 xv6 原有进程、虚拟内存与文件系统框架之上，新增了 Linux 系统调用兼容层、EXT4 轻量级只读解析器以及内置竞赛编排器，形成“双 ABI 共存”架构——同一内核可根据进程类型自动分发到 xv6 原生或 Linux 系统调用路径。其 EXT4 读取器完全从零实现，通过 extent 树遍历支持 4K 块读取，并借助 xv6 的 buffer cache 完成块设备适配；同时引入 per-process 运行时 tmpfs，在只读镜像上提供文件写入能力。工程层面，项目将测试编排逻辑内置于内核中，利用 exit 系统调用触发测试推进，省去外部脚本与内核交互的复杂性；所有未实现的系统调用采用“返回零而不报错”的占位策略，避免测试程序因 ENOSYS 异常退出。在约三周开发周期内，四人团队凭借精确的需求对齐和务实的“够用就好”设计，成功获得 312 分并通过初赛，验证了目标驱动型内核开发的有效性。
