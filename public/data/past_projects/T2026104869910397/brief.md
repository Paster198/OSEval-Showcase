CoreCraft 是一个基于 Rust 语言实现的宏内核操作系统项目，面向 RISC-V 64 (SV39) 和 LoongArch 64 双架构，具备较高的工程完成度和 Linux ABI 兼容性。

该项目共包含约 87,000 行 Rust 源码，采用 Cargo workspace 组织为 13 个子 crate。内核采用高半虚拟地址空间设计，通过 `ArchInterface` trait 实现干净的架构抽象，将硬件细节与内核核心逻辑解耦。内核启动引导、中断/异常处理、页表操作和上下文切换在两种架构上均完整实现，LoongArch 端还特别包含非对齐访存的软件模拟。

内存管理方面，项目实现了写时拷贝 (COW)、延迟分配和策略-执行分离的页面故障处理，确保了虚拟内存语义的正确性。虚拟文件系统 (VFS) 通过统一的 trait 体系集成了 ext4（基于 lwext4 C 库的 FFI 封装）、procfs、devfs 和 tmpfs 等多个后端，管道与块缓存层均具备完整的 POSIX 语义。信号子系统覆盖了标准信号、实时信号队列、信号跳板和同步信号强制投递等完整语义。

系统调用层定义了 237 个调用，囊括文件 I/O、进程管理、内存管理、信号和定时器等核心领域。项目内置嵌入式测试运行时，支持声明式测试配置，并与 Linux Test Project (LTP) 深度集成。主要短板在于单核设计（无 SMP 支持）和网络协议栈仅实现了 AF_ALG 加密 API socket，缺乏 TCP/IP 支持。整体而言，该项目在双架构支持、VFS 设计和内存管理正确性方面表现突出，是一个教学与竞赛背景下质量较高的 Rust OS 内核实现。
