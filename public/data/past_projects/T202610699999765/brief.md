NPUcore-Ovo 是一个面向操作系统竞赛场景、从零开发的 Rust 语言内核，同时支持 RISC-V 64 与 LoongArch 64 双指令集架构。该项目以 Linux ABI 兼容为核心目标，构建了统一硬件抽象层，隔离上层子系统与底层页表、异常处理及多核启动差异。内核实现了完整的三级调度框架（RT 与 CFS 实时/分时调度）、基于 sv39 和 LAFlex 的虚拟内存管理（含 CoW、ZRAM 内存压缩及磁盘交换），并全量自研 EXT4 extent 树文件系统，兼顾 FAT32 只读识别。信号处理、futex 同步、定时器与超过 200 个系统调用使内核具备运行 BusyBox 等复杂用户程序的能力。多核 SMP 启动与 per-CPU 调度进一步提升了硬件的利用水平。项目在调度算法、页缓存、内存压缩以及进程凭证建模等方面体现出较强的工程积累，但对高级 Linux 特性（如命名空间、cgroups、网络设备驱动）的支持仍较初步，整体在竞赛内核中属于高完成度作品。
