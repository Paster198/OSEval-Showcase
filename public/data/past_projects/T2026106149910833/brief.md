OSKernel2026-X 是一个基于 Rust 语言实现的类 Unix 单体内核，面向全国大学生计算机系统能力大赛操作系统内核实现赛道。项目采用分层单体内核架构，通过独创的 12 个架构契约模块实现硬件抽象，以函数指针和 BoundaryMode 三阶段状态机将平台依赖参数化，支持 RISC-V 64 和 LoongArch64 双架构。内核规模约 84,000 行 Rust 代码，零外部依赖，实现约 150 个 Linux 系统调用，覆盖进程管理、信号处理、内存管理、文件 I/O、IPC 及时钟等核心功能。调度器采用 EDF 实时调度算法，支持 20 种阻塞等待通道和 futex requeue 机制。文件系统层提供 ext4 只读解析、内置 proc/dev 伪文件系统和内存可写覆盖层，能够运行动态链接的用户态 ELF 程序。项目充分利用 Rust 类型系统的 const fn 编译期验证、状态机模式等特性，在 170 个源文件、双架构目标上实现零警告构建，展示了高水平的工程实现质量。
