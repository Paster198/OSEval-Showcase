Explosion OS 是一个基于 Rust 语言编写的宏内核操作系统项目，主要面向操作系统竞赛与底层系统教学场景。该项目在 rCore-Tutorial 的基础上进行了大幅扩展，技术定位于构建一个功能广度高、具备多架构支持潜力的现代操作系统内核。其核心特点包括实现了约 75 个 Linux 兼容系统调用，涵盖进程与线程调度、SV39 虚拟内存管理、虚拟文件系统以及基础网络协议栈，并采用基于 trait 的硬件抽象层设计以隔离 RISC-V 与 LoongArch64 的架构差异。项目的核心亮点在于从零自研了近 7000 行的 EXT4 文件系统，支持 extent 树与完整的块分配机制；同时实现了写时复制 Fork、mmap 内存映射、延迟浮点上下文保存等高级特性，并成功集成 BusyBox 以支持 Shell 脚本执行，展现了扎实的系统底层设计与工程实现能力。
