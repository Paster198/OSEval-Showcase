StarryOS 是一个基于 ArceOS unikernel 组件化框架构建的 Linux 兼容型宏内核，完全采用 Rust 语言开发，主要面向操作系统比赛场景，支持 RISC-V、LoongArch、x86_64 和 AArch64 四种架构。项目在约 2.3 万行 Rust 代码中系统性地实现了进程管理与调度、含写时复制的虚拟内存管理、支持 ext4 与 FAT 后端的文件系统框架、基于 smoltcp 的 TCP/UDP/Unix 套接字网络栈、完整的 TTY 与伪终端子系统、信号处理、futex 同步机制、SysV 消息队列与共享内存、以及 epoll 等 I/O 多路复用设施，共计 239 个系统调用。内核能够同时运行 glibc 和 musl 编译的用户态程序，在 RISC-V 平台通过 506 个 LTP 系统调用测试用例，Busybox、Lua、iozone、iperf 等应用均可稳定运行。其核心亮点在于将本用于单地址空间 unikernel 的框架逆向构造为支持多进程的宏内核，并以标准测试套件为驱动的增量式工程方法，配合 Rust 类型系统与作用域局部存储等机制，实现了安全、紧凑且兼容性持续增长的内核实现。
