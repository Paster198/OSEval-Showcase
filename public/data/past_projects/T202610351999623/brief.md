OSKernel 是一个基于 Rust #![no_std] 环境构建的 RISC-V 64 位 UNIX-like 微内核，旨在以精简的代码规模提供足以运行 busybox 与 musl/glibc 双 libc 用户态工具链的实用能力。内核采用回调注入模式将架构层与调度、系统调用等上层策略完全解耦，所有全局资源池以固定数组实现零堆分配，消除了内核堆碎片化隐患。其 Sv39 虚拟内存子系统支持独立用户地址空间、按需缺页处理、mmap 及 COW 隔离；进程模型覆盖 fork/exec/waitpid/vfork/CLONE_VM 线程及完整的 POSIX 信号投递与用户态信号帧机制。文件系统方面，内核提供了 RamFs、DevFs、ProcFs 和具备读写能力的 Ext4 驱动，Ext4 实现了 extent 树遍历、block/inode 分配器和元数据操作。系统调用实现了 70 余个 Linux RISC-V ABI 兼容调用，覆盖 I/O、进程、文件、信号、内存和时间等类别。亮点包括 musl 与 glibc 双动态链接支持、SBI Timer 活性探测以解决固件兼容缺陷，以及清晰的上下文安全校验与运行时断言体系，整体展现了成熟的系统工程实践。
