REMOS 是一个基于 xv6-riscv 深度改造的 RISC-V 64 位宏内核操作系统，面向 QEMU 与 VisionFive 平台。项目使用 C 语言编写，实现了约 50 个系统调用，涵盖进程、内存、文件与时间管理等核心功能。其核心特点是对底层子系统进行了大幅重构：内存管理采用 Buddy 算法，引入每 CPU 独立内存池与跨池窃取机制以优化多核性能；虚拟内存引入 per-process 内核页表，显著提升内核访问用户内存的效率。项目的最大亮点是成功移植并集成 lwext4 库，使系统具备了支持日志恢复、Extent 管理与目录索引等高级特性的完整 ext4 文件系统。总体而言，REMOS 在文件系统与内存分配的工程集成上表现优异，是一个功能完备且具备架构优化深度的竞赛级内核。
