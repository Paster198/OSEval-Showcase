oskernel 是一个基于 Rust 语言开发的类 Unix 宏内核，起源自 rCore-Tutorial-v3，目标平台为 RISC-V 64 位（QEMU virt），并针对 LoongArch 64 位进行了最小启动验证。内核采用 Sv39 虚拟内存与单核 FIFO 协作式调度，在内存管理、进程管理、文件系统和网络协议栈等关键子系统上均实现了较完整的功能覆盖。其核心特点是双文件系统后端自动切换：既可操作轻量级 easy-fs 读写文件系统，又能通过只读 EXT4 加内存 overlay 层直接挂载标准 EXT4 磁盘镜像，使得用户程序在无写回磁盘的条件下获得“可读写”体验。系统调用接口兼容约 85 个 Linux 标准调用号，能够运行大量 Linux 用户态测试程序。项目还内置了一套在内核启动后自动执行的测试编排框架，支持 glibc/musl/lmbench 等多种业界基准套件的自动化矩阵测试。相较同类教学或竞赛内核，该项目在工程化集成、EXT4 兼容和自动化验证方面展现出明显特色。
