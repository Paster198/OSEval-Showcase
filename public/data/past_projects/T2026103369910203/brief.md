DDUOS (NighthawkOS) 是一个面向操作系统竞赛的、以 Linux ABI 兼容为目标的现代化异步内核。项目使用 Rust 语言实现，总计约 6.3 万行代码，核心创新在于全异步架构：自研异步运行时将用户进程、内核线程和 I/O 操作统一为 Future 模型，通过工作窃取实现多核负载均衡，天然避免阻塞，简化了传统内核的上下文切换与中断处理逻辑。内核同时支持 RISC-V 64（Sv39 页表）和 LoongArch 64（DMW + 软件 TLB 填充）两种架构，架构抽象层设计精巧，展现了良好的可移植性。功能方面，实现了超过 110 个 Linux 兼容系统调用，覆盖进程管理（fork/clone/execve/wait）、内存管理（完整的 mmap/munmap、写时拷贝）、VFS 层（支持 ext4、FAT32、procfs、devfs、tmpfs 等）、TCP/UDP 协议栈（基于 smoltcp）、以及 epoll、eventfd、inotify、信号等高级 I/O 与进程间通信机制。其深度 ABI 兼容性使得 busybox 等标准用户态程序可直接运行。项目模块化突出，由 22 个独立 crate 组成，构建系统围绕 Docker 环境与离线依赖管理，工程化程度高，是 Rust 语言在操作系统内核领域应用的一个优秀范例。
