MonkeyOS 是一个基于 Rust 语言编写的宏内核操作系统，支持 RISC-V 64 与 LoongArch 64 双架构，主要面向 OSKernel2025 竞赛。该项目通过 polyhal 硬件抽象层实现跨架构兼容，并创新性地利用 Rust 的异步编程模型构建了协作式任务调度器，在保障内存安全的同时提升了内核代码的简洁度。其核心特点包括完善的进程与 COW 内存管理、支持 ext4 等多种文件系统的 VFS 抽象，以及涵盖网络、信号和 System V IPC 的百余项 Linux 兼容系统调用。项目的核心亮点在于实现了 musl 与 glibc 双 C 库的动态链接 ELF 加载，引入了任务模板缓存机制以加速程序启动，并在 LoongArch 架构下完整实现了 PCI 总线枚举与 VirtIO 设备驱动，展现出较强的系统底层工程能力与架构设计水平。
