XJC-OS 是一个基于 ArceOS 组件化框架构建的 Linux 兼容宏内核，面向 RISC-V 64 与 LoongArch 64 架构，致力于在嵌入式及富应用场景下提供广泛的系统调用支撑与大型 Linux 应用运行能力。项目在 ArceOS 的模块化基础上系统性地实现了进程/线程管理、虚拟内存、文件描述符与伪文件系统、信号、管道、epoll、futex、网络协议栈以及 System V IPC 等完整宏内核子系统，路由 268 个系统调用分支，可无失败通过 571 个 LTP 测试用例。其核心亮点包括：将组件化架构与宏内核设计有机融合，突破了传统 ArceOS 的应用范式；实现专用编译型 LTP 测试运行器，从根本上解决 shell 嵌套退出的可靠性问题，显著提升评测计数精准度；通过独立架构配置文件实现多架构安全并行构建；并采用离线 vendor 管理、运行时动态脚本修补及渐进式进程清理等工程策略，保障了构建鲁棒性与运行兼容性。整体而言，XJC-OS 功能完整、结构清晰，在系统调用覆盖度、多架构支持与工程实践方面均展现出较高成熟度。
