RyOS 是一个以 Rust 语言编写、面向 RISC-V 64 和 LoongArch 64 双架构的类 Unix 操作系统内核。项目整体代码规模约 6.6 万行，以 QEMU 为主要运行平台，实现了涵盖内存管理、文件系统、网络协议栈、进程管理、信号处理、SMP 多核支持在内的完整内核功能体系。

其核心特点在于自研程度极高：文件系统方面不仅构建了完整的 VFS 抽象层，还自研了纯 Rust 的 ext4 实现（含 extent 树、HTree 目录索引和 jbd2 风格 WAL 日志）以及支持长文件名的 FAT32 实现；网络栈方面完全从 RFC 规范出发，自研了包含完整 11 状态机、拥塞控制、RTT 估计、零窗口探测等机制的 TCP/IP 协议栈。内核实现了约 180 个系统调用，覆盖文件 I/O、进程管理、信号、futex（含 PI 互斥锁和 robust list）、epoll 事件驱动、System V 共享内存等主要类别。

技术亮点包括：异步优先的内核架构将系统调用和 I/O 统一建模为 async fn；多核 work-stealing 与加权 vruntime 调度相结合实现负载均衡与公平性；利用 RISC-V ASID 实现跨核精确 TLB 击落配合 epoch 延迟帧回收解决多核内存管理难题；以及飞行记录仪、心跳观测、lockdep 等诊断基础设施，体现了扎实的工程实践能力。
