cabbageOS 是一个面向 RISC-V 架构的类 Unix 多核操作系统内核，主要定位于操作系统竞赛与教学实践，支持 QEMU 模拟器与 VisionFive2 硬件平台。其核心特点在于采用 C 语言与汇编构建主体，并集成 Rust 编写的底层设备驱动。系统实现了高度完整的内核子系统，包括基于 Buddy 算法的多核物理内存池与 Sv39 虚拟内存管理，支持按需分页与写时复制；采用 PCB 与 TCB 分离的设计，支持多进程多线程模型及 clone 系统调用；同时构建了包含 FAT32、ext4 和 procfs 的 VFS 虚拟文件系统抽象层。该项目的亮点在于出色的多核内存分配协同设计、高达 95% 的系统调用覆盖率以及约 85% 的整体功能完整度，展现了极高的工程成熟度与架构合理性。
