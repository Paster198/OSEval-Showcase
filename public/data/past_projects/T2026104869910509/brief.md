LetsgOS 是一个基于 Rust 语言开发的宏内核操作系统，追求与 Linux ABI 的二进制兼容，同时支持 RISC-V64 和 LoongArch64 双 CPU 架构。内核整体采用异步模型，将系统调用建模为 Rust Future，借助 async/await 实现协作式任务调度，形成统一的异步 I/O 路径。项目以约六万六千行 Rust 代码实现了约一百九十三个系统调用，覆盖进程管理、内存管理、EXT4/FAT32 文件系统、TCP/UDP/Unix 套接字网络栈以及信号、futex 等核心子系统，并提供了 epoll、eventfd、signalfd、timerfd、inotify、memfd、fanotify、io_uring、BPF 等一系列现代 Linux 特殊文件与机制。在架构设计上，通过条件编译与过程宏实现双架构的统一抽象，内核入口、页表、陷阱处理、TLB 管理等关键路径均针对 RISC-V 与 LoongArch 分别优化；在虚拟内存方面，采用类型化的 VMA 设计与模块化缺页处理函数指针，支持写时复制、按需加载与文件映射。该内核已完成对 dash、busybox 等复杂用户态程序的适配，体现了在竞赛级 OS 内核中较高的 ABI 兼容度和工程完整性。
