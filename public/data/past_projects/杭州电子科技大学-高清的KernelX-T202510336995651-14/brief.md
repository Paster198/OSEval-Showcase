KernelX 是一个基于 RT-Thread Smart 构建的面向 RISC-V 64 位平台的微内核操作系统。其技术定位是在工业级实时操作系统底座之上，通过深度改造实现 Linux 系统调用兼容，从而支持标准用户态程序的运行。该项目的核心特点在于构建了包含约 80 个系统调用的 Linux ABI 兼容层，完整涵盖了进程管理、内存管理、POSIX 信号、Futex 同步及 VFS 文件系统等关键子系统。其技术亮点包括：将 clone 与 fork 语义统一并严格对齐 Linux RISC-V 64 位参数传递规范；引入双变量设计修正了 brk 堆管理语义；修复了 openat 路径拼接等底层缺陷；并补充实现了 writev、sendfile、mprotect 等高级系统调用。KernelX 以务实的工程路径，在复用成熟基础设施的同时，显著提升了系统的 Linux 兼容性与生态支持能力。
