SystemNQB 是一个基于 xv6-riscv 二次开发的 RISC-V 64位操作系统内核，采用 C 与 Rust 混合编程的创新架构。其技术定位在于探索现代内存安全语言在传统教学级微内核中的工程化落地。核心特点包括：C 语言主导进程调度、虚拟内存与中断处理等核心逻辑，Rust 语言则接管 ext4 文件系统与伙伴系统物理页分配器，并通过自动生成的 FFI 实现双向交互。此外，内核高度兼容 Linux RISC-V 系统调用规范，支持 ELF 程序加载、Shebang 脚本执行以及基于 clone 的线程组管理。项目亮点在于前瞻性的混合语言设计，不仅利用 Rust 生态提升了底层子系统的安全性，还引入了泛型对象池设计以优化内核资源管理。同时，其高度自动化的构建系统能够自动生成系统调用表与跨语言绑定代码，展现了出色的工程实践与架构设计能力。
