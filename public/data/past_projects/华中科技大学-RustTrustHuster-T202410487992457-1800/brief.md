TrustOS 是一个基于 Rust 语言开发的 RISC-V 64位宏内核操作系统，主要面向操作系统教学与高水平学科竞赛，致力于提供高度兼容 POSIX 标准的系统环境。其核心特点在于实现了 105 个系统调用，构建了包含 SV39 分页、写时复制、按需懒分配及 System V 共享内存的完整内存管理子系统，并集成了基于 lwext4 的 ext4 文件系统与 VFS 抽象层。在进程与同步管理方面，项目支持完整的 clone 线程模型与 Futex 机制，同时实现了符合 Linux 规范的用户栈信号帧与 SA_SIGINFO 信号处理。其技术亮点在于充分利用 Rust 的所有权模型保障内核内存安全，通过辅助向量完美支持用户态动态链接，并具备 QEMU 与 VisionFive2 多板级适配能力，展现了极高的工程模块化水平与系统底层开发质量。
